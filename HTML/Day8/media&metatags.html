Welcome to CinePlayer Hub

CinePlayer Hub is a multimedia webpage created using HTML5 to demonstrate various modern web elements and features.

This webpage includes audio players, video players, embedded YouTube videos, Google Maps integration, semantic HTML tags, non-semantic tags, favicon, meta tags, and Open Graph properties.

It showcases how multimedia content can be added to a webpage using HTML5 elements such as audio, video, iframe, section, header, footer, div, and span tags.

This project is designed as a simple cinema-themed multimedia website to provide an engaging user experience with entertainment content.
