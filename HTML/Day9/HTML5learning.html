Webpage

Definition: A webpage is a single document displayed in a web browser.

Example: Home page, Contact page.

Website

Definition: A website is a collection of webpages connected together.

Type Definition Example
Static Website Content remains same Portfolio Website
Dynamic Website Content changes Amazon, Instagram

HTML

Definition: HTML stands for HyperText Markup Language. It is used to create webpages.

Example: <h1>Welcome</h1>

DOCTYPE HTML

Definition: It tells browser that the document uses HTML5.

Example: <!DOCTYPE html>

Tags

Definition: Tags are keywords inside angle brackets.

Example: <p>

Elements

Definition: Elements contain opening tag, content and closing tag.

Example: <p>Hello</p>

Definition

Data projection means displaying information to users in a clear and organized format. Tables are used to show data in rows and columns.

Meta tags provide information about the webpage to browsers and search engines. They are placed inside the head section of an HTML document.

Meta Tag Use
